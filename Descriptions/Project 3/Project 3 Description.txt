Few weeks back we pulled out an old fireplace. Removed the drywall and added built in coaxle, power cables and Ethernet cables behind the now cement board so that we could get a wireless look compared to wires hanging everywhere and visible. 
We started by fishing wires below and through the existing walls and managed to not have to open or disrupt any unnecessary drywall
We then had a special order stone shipped in and hand built this mantel so that it could hold up against the larger surround sound bar. 
We were able to line everything up and make it seemless and are now tearing out the stairs and re doing the rails so that it all matches within the house.
